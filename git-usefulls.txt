Start rebase:

In the editor, change only the first two lines from pick to edit, save, close.

When Git stops at commit 1, run:
GIT_COMMITTER_NAME="Your Name" GIT_COMMITTER_EMAIL="new@email.com" git commit --amend --no-edit --author="Your Name new@email.com"
git rebase --continue

When it stops at commit 2, run the same amend command again, then:
git rebase --continue

git log --format="%h %an <%ae> | %cn <%ce>" --reverse -n 5
